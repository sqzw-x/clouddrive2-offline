# CloudDrive2-Offline

> 嗅探磁力链接并一键添加到 clouddrive2 离线下载

## 安装

[GitHub Release](https://github.com/sqzw-x/clouddrive2-offline/raw/release-nightly/bilibili-gate.mini.user.js)

## 功能

- 磁力链接嗅探: 自动检测页面中的磁力链接并添加 `离线下载` 按钮
- 离线任务管理: 支持查看离线任务状态, 删除离线任务 (与 CloudDrive2 内置管理器相似)
